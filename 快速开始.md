# 🚀 快速开始 - 剪贴板历史工具

## 📦 安装（3 步完成）

### 1️⃣ 安装应用（30 秒）

```
双击 DMG 文件
     ↓
拖动到「应用程序」文件夹
     ↓
右键点击应用 → 选择「打开」
```

### 2️⃣ 授予权限（1 分钟）

```
打开「系统设置」
     ↓
隐私与安全性 → 辅助功能
     ↓
点击 🔒 解锁 → 添加「剪贴板历史」
     ↓
勾选应用 ✅
```

### 3️⃣ 开始使用（立即）

```
应用会出现在状态栏 📋
     ↓
复制任何内容（⌘C）
     ↓
按 ⌘⌥V 打开历史（屏幕底部显示）
     ↓
用 ←→ 选择内容 → 按回车粘贴 ✨
```

---

## 🎯 核心功能

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 打开历史 | `⌘⌥V` | 在屏幕底部显示剪贴板历史 |
| 选择项目 | `←` `→` | 左右移动选择（横向滚动） |
| 粘贴内容 | `回车` | 自动粘贴到输入框 |
| 关闭窗口 | `ESC` | 关闭历史窗口 |
| 删除项目 | `Delete` | 删除选中的历史 |
| 搜索过滤 | 直接输入 | 实时搜索历史 |

## ⚡ 常用操作

### 批量复制
1. 连续复制多个内容（⌘C）
2. 应用自动保存每一个
3. 最多保存 200 条

### 快速粘贴
1. 按 `⌘⌥V` 打开（屏幕底部横向显示）
2. 用 `←→` 左右选择项目
3. 按 `回车` 粘贴

### 搜索历史
1. 打开历史窗口
2. 直接输入关键词
3. 实时过滤结果（横向排列）

## ❓ 快速问答

**Q: 快捷键冲突怎么办？**
A: 当前版本暂不支持自定义，未来版本会添加此功能。

**Q: 历史记录会同步到其他设备吗？**
A: 不会，所有数据仅保存在本地。

**Q: 支持哪些 macOS 版本？**
A: macOS 11.0 (Big Sur) 及以上版本。

**Q: 应用占用多少空间？**
A: 应用本身 < 5 MB，历史数据取决于保存的内容。

**Q: 安全吗？**
A: 完全安全，所有数据保存在本地，不会上传任何信息。
