# Changelog

## [1.0.0] - 2026-01-08

### New Features
- **New Interface**: Screen bottom display, 100% width, 20% height.
- **Card Layout**: Horizontal scrolling card design.
- **Shortcuts**: Global `⌘⌥V` (Command + Option + V) to open history.
- **Navigation**: Left/Right arrows to select, Enter to paste.
- **Mouse Support**: Click item to paste immediately.

### Improvements
- **Click Behavior**: Restored single-click to paste functionality.
- **Arrow Keys**: Arrow keys only select, do not auto-paste (press Enter to paste).
- **Paste Reliability**: Added buffer time after paste operation to ensure correct content is pasted.
- **Cleanup**: Removed unnecessary settings switches.

### Documentation
- Updated README, User Guide, and Quick Start documents.
