# 🎉 更新说明 - 剪贴板历史工具 v1.0.0

## 📅 更新日期：2026-01-08

---

## ✨ 重大更新

### 1️⃣ 快捷键变更

**旧版本：**
- `⌘⇧V` (Command + Shift + V)

**新版本：**
- `⌘⌥V` (Command + Option + V)

> 💡 **为什么改变？**  
> Command + Option + V 更容易按，避免与某些应用的快捷键冲突

---

### 2️⃣ 全新界面设计 🎨

#### 旧版本界面：
- ✗ 居中悬浮窗口
- ✗ 固定大小（600x500）
- ✗ 垂直列表布局
- ✗ 上下滚动选择

#### 新版本界面：
- ✅ **屏幕底部显示**
- ✅ **宽度 100%，高度 20%**
- ✅ **横向卡片式布局**
- ✅ **左右滚动选择**
- ✅ **半透明黑色背景**
- ✅ **更大的预览区域**
